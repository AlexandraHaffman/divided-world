# МИСС РАЗДЕЛЁННЫЙ МИР — 2061
## Технический отчёт агента

Автономная симуляция конкурса красоты во вселенной «Разделённый мир». Многостраничный статический сайт в папке `miss-divided-world-2061/`. Данные — единый источник (`data/*.json` + бандл `data/data.js` → `window.DW`), рендер на клиенте, работает локально без сервера и на GitHub Pages.

## Как открыть
- Секретный вход: на странице `characters/index.html` в служебной строке шапки — неприметный глиф **♛** (проявляется золотым свечением при наведении) → ведёт в закрытую трансляцию.
- Пароль доступа: **`корона-разлома`** (в исходнике хранится только SHA-256-хэш; сменить: `printf '%s' 'новый-код' | sha256sum` → заменить `ACCESS_HASH` в `assets/js/main.js`).
- Два режима на главной: «Начать хронику» (последовательное чтение, итоги скрыты) и «Полный архив со спойлерами».

## Формула судейства
- Судейская ложа — 7 мужчин-лидеров держав: Гилберт (Единая Америка, председатель, объективен), Остерман (Тенебрион), Уолш (Forge), Сорель (Ракшасы), Вистнер / Дорн / Грей (Аркадия). Средний балл раунда = среднее 7 оценок.
- Балл судьи = взвешенная оценка релевантных атрибутов раунда (по его зоне внимания) + характерный разброс (±0.35) + **надбавка предвзятости**.
- Надбавка предвзятости (десятые доли балла, детерминирована): симпатия к фракции + личное отношение (родство/любовь) + «работа с ложей» (закулисные усилия Нимфеи и Валентины). Гилберт — с нулевой надбавкой. Модель открыта и задокументирована на странице правил.
- Предварительные веса: представление 15%, интервью 10%, купальник 30%, платье 25%, костюм 20%.
- Агрегация: 20→10 = 65% предвар. + 35% манифест; 10→5 = 45% полуфинал + 25% вопрос + 30% таланты; 5→3 = 40% образ + 35% фотосессия + 25% общий вопрос; корона = 40% образ + 30% общий вопрос + 30% последнее слово.
- Тайбрейк: 1) интервью, 2) фракционный костюм, 3) последний актуальный раунд, 4) решение председателя Гилберта.
- Модель детерминирована (SHA-256-сид характерного разброса судей); одинаковый вход → одинаковый результат. Ни одного слагаемого за славу, политический статус, боевую силу или мораль — но человеческая пристрастность семерых мужчин учтена открыто и честно.

- Данные конкурса: пересобрать `python3 build/generate.py emit` (генератор в `build/`).
